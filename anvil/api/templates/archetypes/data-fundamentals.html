{# Quick Reference #}
Use Datasets when You have curated text you want to edit, filter, or deduplicate. Fine-tuning data, evaluation sets — small to medium collections.
Use Corpora when You want to scan existing code repos, documentation trees, or large directory structures and auto-chunk them optimally.
Combine both Scan with a corpus for discovery, then import into a dataset for curation — get the best of both worlds.
Pro tip Start with the Add Data tab on the datasets page, then work through. The All Data table shows everything at a glance.
