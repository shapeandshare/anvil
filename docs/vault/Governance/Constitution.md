# anvil Constitution

See [CONSTITUTION.md](../../../CONSTITUTION.md) at repo root for the authoritative version.

## Principles

1. **Zero-Dependency Core** — core engine has zero third-party deps
2. **Educational Clarity** — readability over optimization
3. **Seeded Reproducibility** — all training runs deterministic
4. **TDD Mandatory** — tests before code, 100% coverage
5. **Async-First** — async throughout (except core engine)
6. **Implicit Namespace** — PEP 420, `__init__.py` only for exports
7. **Layered Architecture** — Repository → Service → God Class → Routes
8. **Whimsy Without Compromise** — delightful UX, rigorous code
